# Веб-разработка — Семестр 2

Серверная разработка на PHP и базы данных. Переход от простых форм к полноценной панели администратора с CRUD-операциями.

## Обзор задач

| Задача | Тема |
|--------|------|
| 1 | Настройка FTP и деплой (скриншоты) |
| 2 | Конфигурация сервера с PHP |
| 3 | Обработка форм на PHP, базовая валидация |
| 4 | Расширенная обработка форм, куки |
| 5 | Аутентификация пользователей с сессиями |
| 6 | Панель администратора с CRUD-операциями в БД |
| 7 | Расширенная панель администратора с управлением пользователями |

## Стек

PHP · MySQL · HTML · CSS

## Ключевые концепции

- Обработка HTTP-запросов (GET/POST)
- Валидация и санитизация форм
- Управление куки и сессиями
- Подключение к базе данных (PDO)
- CRUD-операции
- Аутентификация пользователей
- Защита от SQL-инъекций
